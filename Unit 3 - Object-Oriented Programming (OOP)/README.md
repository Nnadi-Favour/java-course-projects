Unit 3 – Object-Oriented Programming (OOP)
Student Record Management System

Overview
This project demonstrates the application of Object-Oriented Programming (OOP) principles in Java by developing a console-based Student Record Management System.

The program allows users to manage student records through a simple menu-driven interface.

Features
 .Add new students
 .Update existing student records
 .Display all student information
 .Store records using ArrayList
 .Input validation and basic error handling

OOP Concepts Demonstrated
 .Classes and Objects
 .Encapsulation using private variables
 .Constructors
 .Getter and Setter methods
 .Static variables and methods
 .ArrayList collections
 .Separation of responsibilities across multiple classes
