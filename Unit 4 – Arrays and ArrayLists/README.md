# Unit 4 – Arrays and ArrayLists


## Overview


This project is a console-based Java application that demonstrates the use of arrays and ArrayLists to perform basic stock price analysis. The program processes a collection of stock prices by calculating the average price, identifying the highest stock price, counting the occurrences of a specific price, and computing the cumulative sum of stock prices.


The project highlights the practical differences between arrays and ArrayLists while reinforcing fundamental programming concepts such as methods, loops, and collection manipulation.


## Learning Objectives


This project demonstrates:


- One-dimensional arrays
- ArrayLists
- Methods
- For loops
- Enhanced for loops
- Mathematical calculations
- Searching and counting
- Data processing using collections


## Features


- Calculate the average stock price
- Find the maximum stock price
- Count occurrences of a specified stock price
- Compute the cumulative sum of stock prices
- Display formatted results in the console
- Demonstrate the use of both arrays and ArrayLists in one application
