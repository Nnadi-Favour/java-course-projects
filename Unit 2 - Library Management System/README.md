# Unit 2 – Library Management System

## Overview

This project is a console-based Library Management System developed in Java. It allows users to manage a collection of books by adding new books, borrowing available books, returning borrowed books, and viewing the current library inventory.

The program uses object-oriented programming concepts together with Java collections to organize and manage book records efficiently.

## Learning Objectives

This project demonstrates:

- Classes and objects
- ArrayList collections
- Methods
- Loops
- Conditional statements
- Exception handling
- Input validation
- Menu-driven programming

## Features

- Add new books
- Update book quantities
- Borrow books
- Return books
- Display all available books
- Validate user input
- Handle invalid numeric entries gracefully

## Technologies

- Java
- ArrayList
- Scanner
- Exception handling (try-catch)
