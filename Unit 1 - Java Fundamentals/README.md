# Unit 1 – Java Fundamentals

## Overview

This project is a console-based Java quiz application that tests users on basic computer science and Java programming concepts. The program presents multiple-choice questions, validates user input, calculates the final score, and displays the user's performance at the end of the quiz.

## Learning Objectives

This project demonstrates the following Java concepts:

- Variables and data types
- User input using the Scanner class
- Conditional statements (if and switch)
- String manipulation
- Arithmetic calculations
- Program flow control

## Features

- Interactive multiple-choice quiz
- Input validation
- Automatic score calculation
- Percentage score display
- User-friendly console interface

## Technologies

- Java
- Scanner class
- Switch statements
- Conditional logic
