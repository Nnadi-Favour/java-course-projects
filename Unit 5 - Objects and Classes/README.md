# Unit 5 – Objects and Classes

## Overview

This project is a console-based Java application that implements a **Course Enrollment and Grade Management System** for a university. The application demonstrates the principles of Object-Oriented Programming (OOP) by using multiple classes to manage students, courses, enrollments, and grades through an interactive command-line interface.

The project emphasizes the use of classes, objects, encapsulation, constructors, instance methods, static methods, and static variables while providing administrators with the ability to efficiently manage course enrollment and student grades.

## Learning Objectives

This project demonstrates:

- Object-Oriented Programming (OOP)
- Classes and Objects
- Encapsulation using private variables
- Getter and Setter methods
- Constructors
- Instance methods
- Static variables
- Static methods
- ArrayLists
- HashMaps
- Command-line user interface
- Exception handling and input validation

## Features

- Add new courses
- Add new students
- Enroll students in courses
- Assign grades to students
- Calculate overall student grades
- Update student information
- Update course information
- Track the total number of enrolled students using static variables
- Validate course capacity before enrollment
- Display an interactive administrator menu
- Display formatted results in the console

## Technologies

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- HashMap
- Scanner
- Java Collections Framework

## Files

- `Unit5.java` – Main program containing the administrator interface and menu-driven application.
- `Student.java` – Defines the Student class, including student information, course enrollment, and grade management.
- `Course.java` – Defines the Course class, including course details, enrollment capacity, and static tracking of total enrolled students.
- `CourseManagement.java` – Manages courses, student enrollment, grade assignment, and overall grade calculation.

## How to Run

1. Open the project folder in GitHub Codespaces.
2. Open the terminal.
3. Compile all Java files by running:

```bash
javac *.java
```

4. Run the program using:

```bash
java Unit5
```

5. Follow the on-screen menu to:
   - Add courses
   - Add students
   - Enroll students
   - Assign grades
   - Calculate overall grades
   - Update student information
   - Update course information
   - Display the total number of enrolled students
   - Exit the program
